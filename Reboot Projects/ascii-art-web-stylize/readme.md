# Ascii-Art-Web
    this project takes the users input from the website (localhost) and transforms it into asciiart version of it

# How to run
- In the terminal, type `go run main.go` or `go run .`
- Open your favorite browser and type `localhost:8080`
- In the text field, enter your text
- Choose a banner type / font for your text
- Press the "Submit to function" button
- Done!


Note : if you need to go down a line (new line) use "\n", DO NOT PRESS ENTER as this will cause an error

# Implementation
in this project we used go land for web-hosting/ http handeling, html for the web pages and css to style them.
