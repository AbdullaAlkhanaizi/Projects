# Ascii-Art-Web-dockerize
    this project takes the users input from a dockerized website (localhost:4000) and transforms it into asciiart version of it

# authers

# How to run
- launch the docker desktop application and start the `ascii-art-web-dockerize` image
- at the settings page for the image drop down the more settings menu and pick a port other than `8080` `(ex 4000)`
- Open your favorite browser and type `localhost:8080`
- In the text field, enter your text
- Choose a banner type / font for your text
- Press the "Submit to function" button
- Done!


# Implementation
in this project we used go lang for web-hosting/ http handeling, docker for the image and the container launching it, html for the web pages and css to style them.
