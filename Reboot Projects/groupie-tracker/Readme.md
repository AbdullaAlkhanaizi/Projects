# Project : `Groupie - Tracker`

#### this project is about taking data about "Bands" from an online Json file use it to make a website that renders the taken data into "blocks" for each band.

# Languages used in this project : 
#### ``Golang, HTML and CSS ``

# How to use this project :
- cd into the project directory
- type in the terminal ``go run main.go``
- Hover over the link (http://localhost:8080) then press ctrl (control) + left click for windows/linux or press command + left click for mac 
- Done !

# What we learned in this project:

#### In this procet we learned how to get and manipulate "Band" data from a Json file useing structs in golang, then we rendered only the main part of the data (ID,Image,Name,First album and creation date) for efficiency then when the user clicks on the "More Details" button the website loades the rest of the data for the Band.
