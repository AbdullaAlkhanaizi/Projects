# how to run 
## for the server side
- cd into the directory.
- type in the terminal "go run . [port]" in the brackets type the port number to connect to.
- Done!

## for the client side
- launch a terminal
- type in the terminal "nc ip [port]" in the brackets type the port entered bt the server, and the ip of the host machiene (if hosted on this device type localhost).
- enter your username.
- Done!
