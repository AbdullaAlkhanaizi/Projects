# Ascii-Art-FS
 This project is the exact same as the main Ascii-Art except we take another argument from the user (Banner), this allows the user to select from three different Banner or Font types.
 
# authers  
abdusaeed , aalasfoo , abmarhoon

# How to run
- In the terminal, type `go run main.go [String] [Banner] | cat -e ` or `go run . [String] [Banner] | cat -e `.
- Note: in the banner argument please enter the Banner types with capital letters.
- after entering the arguments press enter.
- Done!

# Implementation
in this project we used go-lang only, we didn't need any other language.
